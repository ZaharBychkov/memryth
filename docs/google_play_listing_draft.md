# MEMRYTH Google Play Listing Draft

## Short Description

Private offline library for thoughts, quotes, excerpts, notes, sources, and topics.

## Full Description

MEMRYTH is a private offline library for meaningful text fragments.

Save quotes, personal thoughts, and longer excerpts in one place. Add context with topics, author, source, and your own note, then return to what matters through search, filters, favorites, and sorting.

MEMRYTH is built for a personal text memory space, not a public social feed and not a cloud note service.

Core features:

- save quotes, thoughts, and excerpts;
- add topics, author, source, and personal notes;
- mark important entries as favorites;
- search by text, author, source, note, and topics;
- filter by entry type, topic, and favorites;
- browse a topic index;
- adjust reading text size and line spacing;
- use light or dark theme;
- export your library to a JSON backup file;
- import a JSON backup file in safe merge mode;
- use the app without an account.

Privacy and data:

- offline-first;
- no required account;
- local app storage;
- manual JSON export and import;
- no cloud sync in the current release;
- no advertising or analytics in the current release.

MEMRYTH starts on Android. Cross-platform expansion depends on demand.

## Store Positioning Notes

Use:

- private offline text library;
- commonplace book;
- thoughts, quotes, and excerpts;
- no account required;
- manual backup and restore.

Avoid:

- cloud sync claims;
- AI claims;
- "second brain" positioning;
- "quotes app" as the main category;
- subscription language while the product remains local-only.

## Data Safety Draft

Collected data:

- No data collected by the developer in the current offline-only release.

Shared data:

- No data shared by the developer in the current offline-only release.

Security practices:

- Data is stored locally on the user's device.
- Users can manually export backup files through Android system sharing.
