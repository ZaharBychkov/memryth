# MEMRYTH: результаты ручного beta-smoke

Дата фиксации: 2026-05-07

Цель файла: сохранить реальные находки с теста на телефоне и разделить их на
beta-блокеры, ближайшие улучшения и отложенные функции.

## Сделать перед beta

- В форме записи убрать внутренний скролл многострочных полей. Скроллиться
  должна вся форма, а основной текст и заметка могут расти по высоте.
- В форме записи у мысли показывать поле `Автор`, без длинного
  `Автор / собеседник (необязательно)`.
- Убрать placeholder-подсказки из полей `Источник` и `Моя заметка`, чтобы после
  фокуса оставался только поднятый label и пустое поле.
- Сделать фильтр типов взаимоисключающим:
  `Все / Цитаты / Мысли / Фрагменты / Избранное`.
- При выборе `Случайный порядок` действительно перемешивать записи заново.
- Долгое нажатие на карточку должно включать массовый режим и сразу выбирать
  эту карточку.
- Массовое удаление добавить только в меню массового режима и только с жестким
  подтверждением: `Удалить N записей? Это нельзя отменить.`
- Сделать меню массовых действий заметнее в режиме выбора.
- Зафиксировать приложение в portrait-ориентации для Android beta.
- Скрыть PIN/biometric из основного UI до решения recovery-модели. Код можно
  оставить, но пользователю нельзя предлагать функцию, которая может закрыть
  доступ к данным без восстановления.
- Дополнить onboarding и экран `О приложении` практическими подсказками:
  темы, вложенные темы через `/`, поиск `#тема`, share-to-app, quick add
  shortcut и Android widget.
- Локализовать Android quick add shortcut/widget на русский.
- Уменьшить лаг слайдеров размера текста и межстрочного интервала: во время
  жеста менять thumb локально, а сохранять настройку после окончания жеста.

## Оставить после beta

- История локальных backup внутри app-private storage: список JSON backup,
  даты, переименование, кнопки `поделиться` / `сохранить`. Сейчас оставить
  системный export/import, чтобы не усложнять Android scoped storage перед beta.
- Recovery-модель для PIN/biometric: только после понятного сценария
  восстановления доступа или явного решения, что пользователь принимает риск.
- Настройка скрытия обучающих подсказок возле тем и в справке.
- Расширение Android widget: случайная избранная запись / resurfacing.

## Не делать сейчас

- Не добавлять быструю кнопку удаления в bulk toolbar. Удаление только через
  меню и подтверждение.
- Не строить свою открытую файловую систему backup-архива в shared storage.
- Не обещать sync/cloud/AI/account до реальной реализации.
